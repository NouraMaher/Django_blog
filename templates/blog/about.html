About My Simple Blog

Welcome to My Simple Blog!

This is a simple blog application built with Django. It demonstrates the core concepts of Django web development including:

  • Models: Post, Category, and Comment models with relationships
  • Views: Function-based views for handling different pages
  • Templates: HTML templates with template inheritance
  • Forms: Django forms for user input
  • Admin: Django admin interface for content management

Features

  • 📝 Create and publish blog posts
  • 🏷️ Organize posts by categories
  • 💬 Comment system
  • 🔍 Search functionality
  • 📄 Pagination
  • ⭐ Featured posts
  • 📱 Responsive design

Technology Stack

  • Backend: Django (Python web framework)
  • Database: SQLite (default Django database)
  • Frontend: HTML, CSS, Bootstrap 5
  • JavaScript: Bootstrap JS for interactive components
